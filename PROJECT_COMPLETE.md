### 1. Local Testing (Now)
```bash
# Open in browser
1. Navigate to: file:///path/to/index.html
   OR
2. Use Python: python -m http.server 8000
3. Visit: http://localhost:8000
```

### 2. Git Setup
```bash
cd "c:\Users\jlpil\OneDrive\Desktop\Multimedia Authoring Website"
git init
git add .
git commit -m "Initial commit: Multimedia portfolio"
```

### 3. Create GitHub Repository
```
1. Go to github.com
2. New Repository
3. Name: multimedia-authoring-portfolio
4. Make it PUBLIC
5. Create
```

### 4. Push to GitHub
```bash
git remote add origin https://github.com/Coco-XD20/multimedia-authoring-portfolio.git
git branch -M main
git push -u origin main
```

### 5. Enable GitHub Pages
```
1. Settings → Pages
2. Source: main branch, / root
3. Save
4. Wait 1-2 minutes
5. Visit: https://Coco-XD20.github.io/multimedia-authoring-portfolio/
```
